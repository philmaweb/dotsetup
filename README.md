#dotsetup
##WARNING: This will overwrite:
* __~/.vimrc__
* __~/.gitconfig__
* __~/.vim/colors__
* __~/.vim/bundle__

##First Steps:
* install git
* install vim
* install curl 
        sudo apt-get install git vim curl -y

##execute:
```bash
git clone https://github.com/philmaweb/dotsetup.git
./setup.sh
```
Solarized colorscheme, Vundle and VIM-Addons will be installed
